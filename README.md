# Billiards Game

A unique and engaging billiards simulation game built in Java, featuring multiple game modes and dynamic gameplay mechanics.

## Game Features

### Multiple Game Modes
1. **Standard Table**
   - Classic billiards layout with four corner pockets
   - Center starting position for balanced gameplay
   - Perfect for learning the game mechanics
   - 200 balls for an exciting challenge

2. **Mini Table**
   - Compact playing field for faster-paced games
   - Unique four-corner start zones
   - Central target pocket
   - 100 balls for quick, intense matches

3. **Grand Hall**
   - Strategic layout with obstacles
   - Corner-to-corner gameplay
   - Various walls creating challenging paths
   - 150 balls for balanced difficulty

### Game Mechanics
- Realistic ball physics with momentum and collision
- Smooth wall interactions with energy preservation
- Dynamic ball distribution in starting zones
- Accurate pocket detection system

### Technical Features
- Built with Java Swing for smooth graphics
- Multi-threaded game engine for optimal performance
- Configurable map layouts via MapConfig system
- Scalable display system adapting to window size

## How to Play
1. Choose your preferred map from the menu
2. Balls will be distributed in the designated starting zones
3. Watch as the balls move and interact
4. Game ends when a ball falls into a pocket
5. Choose to restart or return to menu

## Controls
- Game automatically simulates ball movement
- Interactive menu system for map selection
- Restart option after each round
- Quick return to menu feature

## Technical Requirements
- Java Runtime Environment (JRE) 8 or higher
- Minimum resolution: 800x600
- Windows/Linux/MacOS compatible

## Development
This game demonstrates advanced Java programming concepts including:
- Object-oriented design
- Physics simulation
- Multi-threading
- Event handling
- Graphics rendering

## Project Structure
- `src/`: Source code files
  - `Ball.java`: Ball physics and rendering
  - `BilliardPanel.java`: Main game panel and mechanics
  - `MapConfig.java`: Map configurations
  - `Main.java`: Game initialization
  - `MenuPanel.java`: Menu interface
- `lib/`: Dependencies and libraries
- `bin/`: Compiled class files
